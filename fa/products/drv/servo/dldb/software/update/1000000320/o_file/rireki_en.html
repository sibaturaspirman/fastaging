Improvement history of firmware version E9 (December, 2024)

There are no major improvements.

Improvement history of firmware version E8 (December, 2024)

No. Description
1 Improved the detection accuracy by changing the detection threshold of Output open-phase error (AL. 139.2) for servo amplifiers with a capacity of 2 kW or more.
2 Fixed a problem in which the speed limit value of the graph function and the drive recorder function would be displayed as "0" in MR-J5-_G-LL.
3 Fixed a problem in which the end code would be invalid when "0" was specified in "Number of data value" in ObjectSubIDWriteBlock of the SLMP command.
4 Fixed a problem in which A-axis would be specified even if B-axis or C-axis was specified for the multidrop station No. in the SLMP 3E frame.

Improvement history of firmware version E4 (April, 2024)

No. Description

Improvement history of firmware version E0 (January, 2024)

No. Description

5 Improved the detection accuracy in [AL. 510_Voltage diagnosis error (safety sub-function)].
6 Made improvements so that the initial communication time is shorter in CC-Link IE TSN safety communications.

No. Description
1 Fixed a problem in which the position within one-revolution may have been incorrect when multipoint Z-phase scale was used.

4 Fixed a problem in which the control response may have varied by 4 ms between servo amplifiers when the communication cycle of the servo amplifier was set to 8 ms.
5 Fixed a problem in which TPR1 to TPR3 (Touch probe 1 to Touch probe 3) may have been valid in servo amplifiers not supporting the touch probe function.
6 Fixed a problem in which the power needed to be cycled twice when [PA04.2 Servo forced stop selection] was set from "1" to "0" by the parameter automatic setting.
7 Fixed a problem in which the B-axis of 2-axis servo amplifiers may not have been able to drive the motor when the CC-Link IE TSN communication cycle was set to 62.5 us.
8 Improved the estimation accuracy in the high-frequency band of the machine analyzer function.
9 Fixed a problem in which Safety communication error 1 - Receive data error B6 (safety sub-function) (AL. 581.E) may have been detected at the initial communication of CC-Link IE TSN safety communication of multi-axis servo amplifiers.
