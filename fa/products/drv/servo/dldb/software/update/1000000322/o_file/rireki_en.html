Improvement history of drive system sizing software Motorizer Ver. 1.055H (November, 2024)

No. Description
Tools 1 Added support for creating the FA Integrated Selection Tool format file for inverters and sensorless servos.
To create a file, select "Tools" from the Motorizer menu, and click "Create FA Integrated Selection Tool format file".
To import the selected unit onto Motorizer, click the "MENU" button in the upper left on FA Integrated Selection Tool, and select "Motorizer (Inverter)" from "Load external linkage file".
Load mechanism 2 Modified so that the specifications under "Option" in the load mechanism screen are expanded instead of collapsed when in an initial state.
Selection candidate list 3 Revised the order of selection results displayed on the selection candidate list.
4 Modified to display "Built-in regenerative resistor" under "Regenerative option" even when regenerative power is not generated in servo amplifiers with a built-in regenerative resistor.
5 Modified to display the resistance regeneration converter unit model name under "Converter unit" when a drive unit and a resistance regeneration converter unit combination is required.
6 Revised the model names of HK-KN servo motors and HK-FN servo motors.
Motor selection 7 Revised the encoder resolution of HK-KN servo motors and HK-FN servo motors.

Improvement history of drive system sizing software Motorizer Ver. 1.050C (April, 2024)

No. Description
Selection calculation 1 Added a setting that does not judge the inertia moment ratio and the mass of load ratio.
This can be set from "Threshold setting" in "Selection candidate list".
When this setting is enabled, the judgement of the inertia moment ratio and the mass of load ratio is not made and the result is always OK.
However, when a geared servo motor is used, the judgement of the inertia moment ratio is made even when this setting is enabled.
Result details 2 Made improvements so that the error messages are expanded instead of being collapsed in the result details.
3 Modified the descriptions of MR-J5-G-HS in the result details.
