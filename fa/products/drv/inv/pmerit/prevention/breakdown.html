Trace function

The operating status (output frequency, etc.) immediately before the protection function activates can be stored in the inverter built-in RAM with the trace function.
Stored data (trace data) can be copied to a USB flash drive and imported into
FR Configurator2, facilitating easy trouble analysis at a separate location.

〈Applicable models〉

・A800 / F800

Clock function

Setting the time with the clock function enables the user to specify the
protective function activation time.
The date and time are also saved with the trace data, making the fault analysis easier.
(The clock is reset at power-OFF. It is not reset if the option FR-LU08 is installed.)

〈Applicable models〉

・A800 / F800

FR Configurator2 for further ease of operation

Using FR Configurator2, easy-to-use software assisting anything from setup to maintenance, much more useful functions are available for users.

▪Diagnostics (Fault history)　

Inverter fault history and fault occurrences are displayed together.

▪Graph function

Inverter data is sampled and displayed in a graph.
Trace data can also be displayed in a graph. This allows for log data and abnormalities to be analyzed.

▪Life diagnosis　

Parts life information read from the inverter is displayed.
A warning icon is displayed in the part life alarm field of the parts recommended for replacement.
