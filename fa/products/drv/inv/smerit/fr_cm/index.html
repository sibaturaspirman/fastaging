Mobile app for inverters

FR Configurator Mobile

Check the occurrence of an error with the app, and quickly identify the problem.

[Compatible inverters]
FR-A800-E／FR-F800-E／FR-E800-E／FR-E800-SCE

Feature

Inverters can be checked and operated
using smart phones or tablets.

The app displays inverter errors.
The display is linked to the error information list of the Mitsubishi Electric FA Global Website, enabling quick checking of the description,
cause, and action without searching.
This will shorten the analysis time when a problem occurs.

Other useful functions are also available.

  • Set parameters

  • Recognize inverters

  • Monitor inverters

  • Set frequencies

  • Set dashboard properties

[Supported products]

FR-A800-E, FR-F800-E, FR-E800-E, and FR-E800-SCE

[Supported environment]

Device: Android (smartphone/tablet), iPhone, and iPad
Operating system: Android 6.0 or later, iOS 11.0 or later,
iPadOS 13.0 or later
Language: Japanese, English, and Chinese (Simplified)
Display: Dark mode (dark theme) is not supported.
