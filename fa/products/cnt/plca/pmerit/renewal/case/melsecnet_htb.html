Utilize the existing network cables to build the MELSECNET/H(10) network system

The existing twisted pair cables of the MELSECNET/B data link system can be used to build the MELSECNET/H network system when replacing the AnS/QnAS Series modules or the A/QnA Series modules with the Q Series modules. Modules are replaced without modifying the previously laid network cables. A network system with an even higher speed can also be configured by replacing the twisted pair cables with CC-Link cables.
