QX41Y41P's I/O assignment is the same as that of the A/QnA Series I/O combined module, AH42. This module can be used as the I/O module on the programmable controller side when using A0J2 Upgrade Tool (manufactured by Mitsubishi Electric System & Service Co., Ltd., refer to page 23) to replace the A0J2(H)CPU.
It is not necessary to change the programs when replacing AH42 or A0J2(H)CPU. (Minimize the need to modify programs)
