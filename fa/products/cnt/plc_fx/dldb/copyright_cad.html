[How to use CAD data]

●By downloading CAD data, you accept the following terms and conditions of use.

  • Decompress the downloaded zip-compressed file to use.

    • How to decompress 1): When the OS has the ZIP decompression function

    1. 1.Right-click on the downloaded file.
    2. 2.Select [Extract All...].
    3. 3.Specify the extraction destination and click the [Extract] button to decompress.

    How to decompress 2): When the OS does not have the ZIP decompression function

    1. 1.For users who do not have a ZIP-compatible decompression tool, please obtain the tool.
    2. 2.Use the decompression tool to decompress.

    *Use of the decompression tool is entirely at user's own risk.

  • How to name a decompressed CAD data file

    • "Product model (single-byte alphanumeric characters (lowercase))" + "_File version" + ".Extension"

    <Example>

    ■MELSEC iQ-F series

    For product model: FX5U-32MR/ES, file version: a, data format: dwg/dxf

    File name: fx5u-32m_es_a_dwg/dxf

    ■MELSEC-F series

    For product name: FX3U-16MR/ES, file version: a, data format: dxf

    File name: fx3u-16m_a_igs_dxf_wmf

  • A file name does not necessarily match the product model.
    For products with the same outline geometry, common files are used for the same CAD data product.

    • <Example>

    Products with the same outline Zip-compressed file name Decompressed CAD file name
    FX5U-32MR/ES fx5u-32m_a_igs.zip fx5u-32m_a_igs
    FX5U-32MT/ES
    FX5U-32MT/ESS
    FX5U-32MR/DS
    FX5U-32MT/DS
    FX5U-32MT/DSS
    FX3G-24MR/ES fx3g-24m_a_igs.zip fx3g-24m_a.igs
    FX3G-24MT/ES
    FX3G-24MT/ESS
    FX3G-24MR/DS
    FX3G-24MT/DS
    FX3G-24MT/DSS
  • When the file name of the decompressed CAD data is different from the selected product model (products with the same outline), change the name to the selected product model (single-byte alphanumeric characters (lowercase)).
●CAD data file format
  • CAD data is provided in the following data formats.
    • CAD format Extension Version
    2D DWG .dwg 2000
    DXF .dxf
    3D* IGES .igs 5.2, 5.3
    Parasolid Text .x_t 10, 18
    STEP .stp AP203

    *Select the CAD format in accordance with your operating environment.

[Precautions]

  • The geometry data is partially simplified in detail to reduce the capacity.
  • Unnecessary geometry, such as chamfering, fillet, draft angle, thread part, and screw head, for considering the layout is omitted.
  • CAD data models should only be used for design consideration.
  • The content of CAD data may be changed without notice due to product improvements.
