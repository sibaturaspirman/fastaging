PLC standard counters can count at some 10 Hz maximum, but using FX PLC built-in high speed counters, high speed pulse inputs can be counted.

Built-in high speed counter numbers and allocation

Built-in high speed counters use general inputs X000 to X007. Input method and input terminal numbers are determined based on whether the counter uses 1-phase or 2-phase, and start or reset functions.

[Examples]
By using C238 (1-phase 1-count), X003 automatically becomes high-speed input terminal.
Also, by using C252 (2-phase 2-count), X000 automically becomes A phase input,
X001 becomes B phase input, and X002 becomes reset input.

FX3U, FX3U PLC counter modes

2-phase 1 edge count counter can be used as a 4 edge count counter by combining a special auxiliary relay (M8388, M8198, M8199).
