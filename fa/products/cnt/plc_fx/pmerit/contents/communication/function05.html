MELSEC-F series Computer Link

  • 製品情報

Features

Data link can be made between computer as master station and multiple PLCs.
Because data link is performed by commands sent from the computer, this is suitable for management and control systems where the computer is the main controller.

  1. 1.Using RS-485 communication devices, up to 16 FX and A PLCs can be connected and send/receive data according to commands from a computer.
  2. 2.Using RS-232C communication devices, data can be sent/received 1:1 with a computer.
  3. 3.Protocol supports PLC's computer link dedicated protocol (formats 1 and 4).
