MELSEC-F series Non-protocol communication

  • 製品情報

Features

MELSEC iQ-F Series modules can communicate with printers, code readers, measurement instruments, etc. having the interface in accordance with RS-232C/RS-485 (RS-422). Communication is performed using sequence programs.
