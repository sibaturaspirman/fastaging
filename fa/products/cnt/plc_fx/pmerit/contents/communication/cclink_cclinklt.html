Other station access function

Other station access function

By connecting GX Works2 to FX3 series main unit master station, program write, device monitor, and maintenance is possible to FX3 series connected as intelligent device station.

Settings for Network parameter

In GX Works2, parameters can be easily set for CC-Link settings and remote device station initial settings, and network diagnosis can be performed.

Expansion of control points

By connecting FX3G(C)/FX3U(C) series on CC-Link, number of I/O control points can be expanded.

  • FX3G/FX3GC　128 pts→256 pts
  • FX3U/FX3UC　256 pts→384 pts
