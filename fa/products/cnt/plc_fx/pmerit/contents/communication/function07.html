MELSEC-F series Peripheral equipment communication

Features

Using RS-422, USB, or RS-232C communication devices, the connection port for peripheral devices can be added.
Devices for sequence programming including a personal computer and display units (GOT) can be connected to the added port.

Concurrent use of peripheral device

Connect an programming tool such as PC software to either one of peripheral devices to avoid programs from being changed by multiple peripheral devices.
