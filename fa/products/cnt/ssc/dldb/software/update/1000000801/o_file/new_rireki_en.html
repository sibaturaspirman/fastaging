New functions added on Motion Control Software SWM-G Ver. 1.102C (March, 2025)

No. Contents
1 Expanded the maximum number of points from 128 to 512 in the 2D pitch error compensation function.
2 Expanded the maximum number of channels from 5 to 32 in the 2D pitch error compensation function.
3 Expanded the maximum number of table points in the E-CAM function from 4096 to 8192.
4 Added a new mode in ServoOnOffSynchronization and improved to enable a more precise servo-on/servo-off synchronization in the synchronization group function.
5 Supports the encoder input for single-turn axis behavior for the E-CAM function.
6 Added the event object condition in event output for the event function.
