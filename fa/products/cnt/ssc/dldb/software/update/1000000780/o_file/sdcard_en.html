Installation procedure from SD memory card

Procedure for extracting a compressed file

  1. 1.Create a folder on the personal computer.
  2. 2.Download and save the file in the created folder.
  3. 3.Right-click the downloaded file and select "Extract all".
    * If the OS does not have a Zip file extraction function, use a ZIP extraction tool to decompress the file.

Installation procedure

  • If software version 08 or later is installed:

You can install Basic System Software and Boot Software at the same time using a bulk installation.



How to batch install the boot software and the basic system software

  1. 1.Delete [instboot], [inst], and [instnew] folders if they exist in the SD memory card root.
  2. 2.Copy the downloaded and extracted [instboot] and [instnew] folders to the SD memory card root.
  3. 3.Turn ON the Motion module power supply while pressing the SD memory card access control switch.
  4. 4.Release the SD memory card access control switch after the POWER LED on the power supply module turn on.
  5. 5.Motion module RUN and PROGRAM RUN LEDs both blink (every 500ms).
  6. 6.When the installation is complete, motion module RUN LED turns off and the PROGRAM RUN and CARD READY LED turn on.
  7. 7.Turn OFF the motion system power supply and ON again.

  • If the software to be installed is version 07 or earlier:

(1) Install Boot Software, then (2) Basic System Software and each add-on library.

To update your motion module from version 04 or earlier:


Precautions

  • Clear the built-in memory after updating to version 05 or later.
  • It may take some time to complete a low-level format of the drive (approximately 5 minutes).
  • Do not turn OFF the motion system power supply while clear and backup/restoration are being executed.
  • When the clear fails, "Drive Error" (error code: 3207H) may occur. In this case, execute the clear again (from step 5).
  • If backup data restoration fails, select [Online] ⇒ [Current Connection Destination] from the menu in GX Works3, and specify a larger value for Time Out (Sec.) on the "Specify Connection Destination Connection" window.

The built-in memory must be cleared after updating the software from version 04 or earlier to version 05 or later.
The built-in memory clear deletes the following data including the absolute position data. Back up and restore data as required before and after the built-in memory clear.

  • User drive (/rom) storage data
  • Latch drive (/lch) storage data
  • Security information
  • A part of system drive (/sys) storage data
  • RAM drive (/ram) storage data

The built-in memory clear and data backup/restoration are executed on the motion control setting.


Procedure for built-in memory clear and data backup/restoration
(If data backup/restoration is not necessary, skip step 2, 3, 4, 8, and 9)


  1. 1.Switch the PLC CPU module to STOP.
  2. 2.Select [Online] ⇒ [Backup/Restore] from the menu on the motion control setting to display the "Backup/Restore" screen.
  3. 3.Click the [Backup] button, and then click the [Yes] button to execute data backup.
  4. 4.After the backup is completed, check that the "Newest backup data acquisition time" is updated on the "Backup/Restore" screen, and then close the "Backup/Restore" screen.
  5. 5.Select [Online] ⇒ [Remote Operation] from the menu on the motion control setting to display the "Remote Operation" screen.
  6. 6.Select [Clear] and click [Execute] to perform the built-in memory clear.
  7. 7.RUN LED will flash while the clear is being executed. After normal completion, RUN LED turns on.
  8. 8. Select [Online] ⇒ [Backup/Restore] on the motion control setting to display the "Backup/Restore" screen.
  9. 9.Click the [Restore] button, and then click the [Yes] button. The data backed up in step 3 will be restored.
