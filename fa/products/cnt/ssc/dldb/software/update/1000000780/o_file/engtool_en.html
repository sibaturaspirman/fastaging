Installation procedure from engineering tool (Motion control setting)

Installation procedure

If your motion module is version 16 or later, The basic system software and the boot software can be can be installed simultaneously by engineering tool (Motion control setting (version "1.020 W" or later)).


How to batch install the boot software and the basic system software

  1. 1.Turn on motion system.
  2. 2.Start the GX Works3 Motion control setting.
  3. 3.Select [Tool] ⇒ [Install Motion Software] to open the "Install Motion Software" window.
  4. 4.Select the radio button "Batch install with motion system software(boot software and motion software package)."
  5. 5.Select the downloaded zip file and execute the installation.
    (Press the “Browse ” button > select the downloaded zip file (sw12dnn-rmtufw_□.zip) > press the" Execute " button)
  6. 6.Wait until the "The installation was completed." dialog box appears.
  7. 7.Turn OFF the motion system power supply and ON again.

If your motion module is version 14 or earlier, you cannot install using engineering tool (Motion control setting).
Use an SD memory card to install. [If installing from the engineering tool (Motion control setting)]
For detailed installation instructions, refer to the MELSEC iQ-R Motion Module User's Manual (Application)
