Improvement history from motion control setting Ver. 1.055H to 1.060N (December, 2024)

No. Contents

5 Solved a problem whereby exported operation profile data would have a wrong profile ID.

7 Solved a problem whereby Motion Control Setting might not start properly if updated from version 1.037P or earlier to version 1.055H.
