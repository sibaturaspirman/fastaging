New function list

New functions added in motion control setting Ver. 1.060N (December, 2024)

No. Contents
1 The unit display for graph on the Operation Profile Data (rotary cutter) screen now changes with the number of cutters.
2 Changed the number of characters a tooltip can display on the Condition Signal Setting and Skip Signal Setting screens.
3 Added condition signal and skip signal settings of multiple axes positioning data as a target for parameter check.

New functions added in motion control setting Ver. 1.055H (July, 2024)

No. Contents
1 Old projects that meet certain conditions* can now be opened with the new Motion Control Setting without adding data of latest functions or extension functions to them.
* All of the following conditions must be met:
- The project has been converted.
- Both the project and Motion Control Setting use the same MCFB versions.
- The project was saved by version 1.055H or later.
2 Object data settings can now be changed in the Axis Parameter Setting screen according to the drive unit used.
3 The Synchronous Section Acceleration Ratio for operation profile data (rotary cutter) can now be set within the following range:
Before: -0.5% to 0.5%
After: -50% to +50%
4 The following functions are supported by operation profile data (multiple axes positioning data):
- Setting skip signals
- Adding PROFILE_POSITIONING_DATA_EXTENDED structures

New functions added in motion control setting Ver. 1.050C (December, 2023)

No. Contents
1 The following functions are supported by operation profile data (multiple axes positioning data):
- Setting jump condition signals
- Jumping to positioning data
2 GX Works3 projects of security version "2" and GX Works3 projects of safety CPUs can now be used in the Logging Setting screen.
3 Motion system software version 30 is supported by the simulator function.
4 A "parameter check" is now performed after users create a new item, apply parameters to each screen, execute conversion, or write data online.
(Whether or not to perform a check after conversion or online writing can be changed in Options.)
