New function list

New functions added to firmware Ver. 1.005 (October 2024)

No. Contents
1 Support for Link device external signal assignment function.

New functions added to firmware Ver. 1.004 (May 2024)

No. Contents
1 In the CC-Link IE TSN Configuration window, support for displaying Communication Period Interval (Min.) and Cyclic Transmission Time (Min.).
2 In advanced synchronous control, support for synchronization encoder axes via link devices has been added.
3 In the mark detection function, support for input signals from link devices has been added.

New functions added to firmware Ver. 1.003 (October 2023)

No. Contents
1 Supports the following encoders as synchronous encoders via servo amplifiers for advanced synchronous control.
- Mitsubishi Electric High Speed Serial Communication Linear Encoder (Incremental Type)
- ABZ Differential Output Type Rotary Encoder (Incremental Type)
- ABZ Differential Output Type Linear Encoder (Incremental Type)
2 Supports following control modes during servo OFF.
- Speed control mode -> Torque control mode
- Torque control mode -> Speed control mode

New functions added to firmware Ver. 1.002 (April 2022)

No. Contents
1 "[Pr.123] Manual pulse generator speed limit value" is supported.
2 "[Pr.442] Cam axis length per cycle change setting" is supported.
3 CC-Link IE TSN protocol version 2.0 is supported.
4 Communication speed 100Mbps is supported.

New functions added to firmware Ver. 1.001 (October 2021)

No. Contents
1 Supports using the Motion module and the FX5 safety extension module together.
* When using the following intelligent modules in addition to the combination of the FX5 safety extension module and the Motion module, use the following versions.
· FX5-40SSC-G/FX5-80SSC-G: "1.001" or later, FX5-DP-M: "1.001" or later,
· FX5-20PG-P: "1.011" or later, FX5-20PG-D: "1.011" or later"
2 Supports firmware update.
* Use CPU module version "1.240" or later.
3 Supports external signal high-accuracy input in the mark detection function.
4 Supports the SLMP frame send instruction (G(P).SLMPSND).
5 Supports the automatic updating of saved parameters of the servo parameter transfer function.
* Use CPU module version "1.250" or later.
6 Supports connection with MR-J5D_-_G4.
