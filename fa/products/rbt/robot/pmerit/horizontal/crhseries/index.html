RH-CRH series introduction

Compact arm for space saving and Lightweight robot arms.

Maximum payload: 3kg, 6kg type

  • Model: RH-3CRH/RH-6CRH
  • Overall height:
    RH-3CRH : 500mm/RH-6CRH : 520mm
  • Robot weight:
    RH-3CRH : 14kg
    RH-6CRH6020 : 17kg
    RH-6CRH7020 : 18kg

Maximum payload: 10kg, 20kg type

  • Model: RH-10CRH/RH-20CRH
  • Overall height:
    RH-10CRH : 560mm/RH-20CRH : 774mm
  • Robot weight:
    RH-10CRH60XX/70XX*1 : 20kg
    RH-10CRH80XX*1 : 21kg
    RH-20CRH8542 : 54kg
    RH-20CRH10042 : 57kg
    (*1 : XX= 20 or 30)

Features

Reducing overall height for a compact design
  • Can be installed in a limited space.
  • Can be incorporated into a wide variety of systems.
  • Easy installation because of simple and lightweight.
High-speed operation and high performance for productivity.
  • CC-Link IE Field Basic as standard.
  • Productivity enhanced due to improved continuous operation performance.
  • Intelligent functions supported by Smart Plus. (*2)
    （*2: Not supported for RH-10CRH/20CRH）
Batteryless motor (RH-10CRH/20CRH)
  • Eliminates the need for periodic battery replacement
  • No risk of losing the origin due to a flat battery
