MELSOFT RT ToolBox3 introduction

Program creation and total engineering support software.

This is computer software to assist with a range of tasks from system startup through to debugging and operation.
The product line includes [RT ToolBox 3 Pro] that allows robot simulations to be run on a 3D CAD software.

  • Main functions
    Creating and editing programs
    Checking the operating environment prior to robot installation
    Simulation of robot operations on the computer
    Estimating cycle times
    Debugging when robots are started up
    Monitoring robots states and faults during operations

The MELFA Works function can be used in RT ToolBox3 Pro.
