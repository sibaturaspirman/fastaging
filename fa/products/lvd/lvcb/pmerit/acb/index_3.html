Electronic Trip Relay (ETR)

A trip relay is an electronic circuit which outputs the trigger signal that activates the ACB open/close mechanism when the trip relay detects applied current that exceeds the set value.

Lineup

Electronic trip relay

General use (WS)

This is a general use protection relay that allows the instantaneous trip characteristics, long time delay characteristics, and other protection characteristics to be configured using dials.

Generator protection use (WM)

This is a generator protection use relay that can be shipped set with the rated current which was specified by the customer. It has an activation characteristics curve that incorporates consideration for generator start-up current.

Special use (WB)

This is a relay for special applications that has only instantaneous trip characteristics. It is designed for use in combination with an external OCR.

Protective coordination use (WF)

This relay allows the long time delay characteristics to be changed in order to achieve coordination with a high-voltage protection relay.

Main relay options

Earth leakage protection (ER)

Earth leakage protection is possible by combining an ETR with earth leakage protection (ER) and an external ZCT.

Ground fault protection (GFR)

By adding the optional setting module G1, protection is possible of arcing ground faults of several hundred amperes.

Extension module (EX1)

This module is necessary when using the extension functions, and performs measurement and setting of various electric quantities.

Display module (DP1, DP2)

This module displays and sets a variety of information including display of circuit information, trip alarm information, and ETR information, as well as configuring the settings of the contact output element installed on the power supply module.
