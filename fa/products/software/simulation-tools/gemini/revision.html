Changelog

Changes from MELSOFT Gemini Professional / Essentials Ver.1.45.00303 to Ver.1.45.00404 (February 2023)

No. Description Supported products
1

<Important>

In the MELFA-BASIC program conversion output function, the problem where positioning data [Rx] and [Rz] were swapped was fixed.

 Professional / Essentials
2

<Important>

The MELFA-BASIC program conversion output function was changed so that the coordinates output are based on the Base Coordinates and Tool Coordinates in the 3D world, and the Base Statement and the Tool Statement in program editor were removed.

 Professional / Essentials
3

<Important>

The MELFA-BASIC program conversion output function was changed so that only Linear Motion Statements which have their ConfigurationMode set to "Fixed" will allow output.

 Professional / Essentials
4 The MELFA-BASIC program conversion output function was changed so that variable name and conditional expression would be output as is. Professional / Essentials
5 The problem where the direction of rotation was different between MELSOFT Gemini and RT ToolBox3 for the axis movement when using Point-to-Point Motion Statements and Linear Motion Statements was fixed. Professional / Essentials
6 The problem where Switch statements set in Gemini and then run in RT Toolbox were executing the default statement as well as the applicable case statement was fixed. Professional / Essentials
