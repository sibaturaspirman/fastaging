Case.04Ensuring prompt support from the Machine Tool Builder when there are problems with machine tools

  • Monitoring
  • Maintenance
  • Cloud integration
  • Individual equipment
Issues
Quickly responding to a problem of a machine tool is difficult because it takes time to identify the cause and take countermeasures. (End users)
Need to remotely check the factory operation status anytime. (End users)
It takes time and a high cost to configure a remote monitoring system and provide a maintenance service by ourselves. (Machine builders)
Solution
  • Remotely access and receive support from the machine tool builder’s call center or Mitsubishi Electric’s service center through cloud server provided by Mitsubishi Electric (For end users)
  • Remotely monitor the operation status of machine tools regardless of location using the dashboard function of iQ Care Remote4U*1*2 (For end users)
  • Easily implement remote service functions on machines equipped with Mitsubishi Electric numerical control using iQ Care Remote4U platform (For machine builders)
  • *1iQ Care Remote4U supports the following models: M800V/M80V series, M800/M80 series, E80 series, C80 series, M700V/M70V series, M700/M70 series, and E70 series.
  • *2The models older than M6 series and C70 series support only the dashboard function.
  • *3The M8V series can be connected without the remote service gateway.
Benefits
  • For end users
    • ・Improve productivity and reduce maintenance costs
      Early grasp of operation status when trouble occurs at anytime from any location. Precise production instructions through information sharing.
    • ・Reduce machine downtime
      Prompt support when trouble occurs. Less downtime with remote diagnosis support.
  • For machine builders
    • ・Easily introduce and operate systems at low cost
      Easy and low-cost introduction of a secure system with cloud. No need for operation and maintenance of dedicated devices and software.
    • ・Reduce after-sales service costs
      Quick/accurate grasp of trouble information. Less troubleshooting time.
