Smart Device Communication Gateway

Use Edgecross basic software*1 to collect production shop floor data and leverage smart device communication gateways*1 and smart device applications*2 to view data on tablets and smartphones. Users can check operating status while moving through the shop floor and conduct system maintenance through AR connectivity.

*1. Software preinstalled on the MI5000. Please refer to the MELIPC MI5000 Series User's Manual (Start-up) for details on preinstalled software.

*2. Smart device applications must be developed by the customer. In addition, development requires smart device application SDK for MELIPC manufactured by International Laboratory Corporation (ILC).
