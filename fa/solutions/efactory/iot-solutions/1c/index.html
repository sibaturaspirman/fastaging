Data Analysis of Installations or Workers

Example

Errors can be detected and their locations can be estimated without stopping the installation. This improves productivity and quality further as well as prevents failure in advance.

*1: When using the user's machine control programs together with the rotary machine vibration diagnosis programs, the scan time will be extended. As this affects the machine control, use multi-CPU configuration.
