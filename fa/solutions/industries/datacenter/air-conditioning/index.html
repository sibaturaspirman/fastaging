Proposals for air conditioning systems

Do you have any requests like…?

Solution : Introduce programmable controller MELSEC iQ-R/iQ-F Series

Solution 1
Ensuring high reliability by adopting duplex system

The MELSEC redundant system ensures the reliability of critical equipment, such as chillers, that cannot be shut down during data center operations. The system is built with general-purpose products, therefore it can be easilymaintained and updated after installation.

Maintenance and update can be performed after installation without stopping the system

  • Tool-less online unit replacement
  • Online addition of slave stations

*Network settings must be completed in advance in the program before using this function.

*1.GENESIS64™ supports duplication of data logging and alarm monitoring.

By connecting a tracking cable capable of extending the distance between the redundant systems to 550 m, the control system and the standby system can be separated. It can be placed far away on the control panel. The optical fiber cable of the tracking cable is not affected by noise and enables high-speed data communication. The system switching time is reduced to 10 ms or less, and high-speed system switching from the control system to the standby system enables continuous control with higher reliability.

Proposals for air conditioning systems

Do you have any requests like…?

  1. 1.Want to eliminate wasteful power consumption
  2. 2.Want to maintain equipment properly before breaks down.

Solution : Introduce the inverter FREQROL Series(energy-saving support device)

Solution
1. Reduces power consumption by inverter operation of pumps and fans
2. Detects machine failures
Solution 1
Reduces power consumption by inverter operation of pumps and fans

The optimal output characteristics (V/F characteristics) can be selected to suit the application and load characteristics, and the inverter automatically streamlines motor control without the need for detailed parameter settings. In addition, it has a built-in auto-tuning function that measures the circuit constant of the motor, enabling stable motor drive even if wiring length is long or differences exist between individual motors. Furthermore, the use of an optimal excitation control mode that enables maximum motor efficiency reduces wasteful motor loss, minimizes power consumption, and achieves good cost-effectiveness.

Solution 2
FREQROL detects machine failure and responds quickly even when a problem occurs

Users can store the relationship between speed and torque without failure, compare the current load condition with the stored failure characteristics, and output an error/warning when out of the normal range. It can detect abnormalities such as filter clogging and belt breakage, making maintenance easier.

Availability of life diagnosis checks is extended as compared to the FR-E700 series. This enhanced diagnosis function ensures reliable operation of the system. The design life of cooling fans and capacitors has been extended to 10 years*1.

*1.Surrounding air temperature: annual average 40°C (free from corrosive gas, flammable gas, oil mist, dust and dirt) Output current: 80% of the inverter ND rating Since the design life is a calculated value, it is not a guaranteed value.

  • *2.Only supports FR-A800/F800/E800 Series
  • *3.Inverter main body terminals ABC
  • *4.Only supports FR-E800 Series

The trace function allows the operating state (output frequency, etc.) immediately before the protective function operation to be stored in the built-in RAM of the inverter. By copying the stored data (trace data) to a USB memory or directly loading it into a PC, trouble analysis is easy with inverter setup software (FR Configurator 2).
